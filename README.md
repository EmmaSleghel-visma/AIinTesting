# AIinTesting

Simple todo list web app for workshop demos.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in terminal (usually `http://localhost:5173`).

## Features

- Add todos
- Mark todos as completed
- Delete todos
- Todos are saved in browser localStorage
